探索專案

程式碼不再只是語法，
而是思維的延伸。

在這次自主學習中，我放棄了傳統死記硬背語法的方式。透過精準的提示詞（Prompt Engineering）與 AI 進行架構討論、邏輯推演與錯誤排除。最大的挑戰不在於寫出程式碼，而在於如何「正確定義問題」並「驗證 AI 的解答」。

實作成果。

三個專案，展現前端互動、邏輯運算與數據分析能力。（點擊下方連結前往網站）

打字練習網 TypingFlow

簡潔介面，可自訂段落長度與休息時間，並記錄階段 WPM，純前端實作。具備新聞模式，可擷取最新新聞作為打字練習素材。運用 JavaScript 監聽鍵盤事件，計算打字速度與準確率，並提供即時反饋。

啟動應用

互動恐怖遊戲 回音：深淵

＊警告 內有突發驚嚇（Jump Scare）場景及氛圍音效，心臟病或癲癇患者不宜遊玩＊。純前端實作的沉浸式體驗。技術上運用 HTML5 Canvas 繪圖與動態 CSS 遮罩製造視覺盲區，結合音效觸發機制增強帶入感。

開始遊戲

Python 數據分析 多維數據對比與預測實驗室

應用程式部署於 Streamlit 伺服器。運用 Python 進行資料清洗與視覺化處理，支援即時動態參數調整，並於網頁端展示柱狀圖、迴歸直線模型與自動化洞察報告。
